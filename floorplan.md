## 三、空間連通關係

### 3.1 開放式區域 (公領域)
| 連通關係 | 說明 |
|---------|------|
| 客廳 ↔ 餐廚區 | **完全開放**，無隔間，視覺通透 |
| 客廳 ↔ 走道 | **開放**，無門，自然過渡 |

### 3.2 私密區域 (私領域)
| 空間 | 門的類型 | 位置 |
|------|---------|------|
| 主臥室 | 統一風格門 (0.85m寬) | 面向走道 |
| 多功能室 | 統一風格門 (0.85m寬) | 面向走道 |
| 衛浴 | 統一風格門 (0.85m寬) | **面向走道** |

### 3.3 門的統一規格 (Japandi 風格)
- **材質**：淺橡木框 + 磨砂玻璃
- **寬度**：0.85m
- **高度**：2.1m
- **類型**：推拉門

## 四、各空間詳細配置

### 4.1 客廳 (8坪) - 座標原點
- **位置**：左上區域
- **尺寸**：5.5m (寬) x 4.8m (深)
- **特點**：
  - 左側大面積落地窗 (2.4m x 2.2m)
  - 與餐廚區開放連通 (右側無牆)
  - 電視牆在北側 (後牆)

**家具配置**：
```
    後牆 (電視牆)
    ┌─────────────────┐
    │   [電視櫃]      │
    │                 │
窗  │     [地毯]      │  → 開放連通餐廚
戶  │      ○茶几      │
    │     ═══沙發     │
    │                 │
    └─────────────────┘
         ↓ 走道
```

### 4.2 餐廚區 (4坪)
- **位置**：右上區域，客廳右側
- **尺寸**：3.5m (寬) x 3.8m (深)
- **特點**：
  - 與客廳完全開放
  - L型廚具靠後牆和右牆
  - 中島兼餐桌面向客廳

**家具配置**：
```
         後牆
    ┌─────────────┐
    │ L型廚具     │
    │ ┌───┐       │
    │ │   │ [中島]│
    │ └───┘  ══   │
    │    [高腳椅] │
    └─────────────┘
      ↑ 開放連通客廳
```

### 4.3 主臥室 (5坪) - 擴大版
- **位置**：左下區域
- **尺寸**：4.0m (寬) x 4.0m (深)
- **入口**：前側牆面，統一風格推門
- **特點**：
  - 後牆窗戶（床頭上方）
  - 衣櫃靠左牆
  - 新增化妝台

**家具配置**：
```
         後牆 (窗戶)
    ┌─────────────────┐
    │ [床頭櫃][床][床頭櫃]│
    │      ════       │
    │                 │
    │[衣櫃]   [化妝台]│
    │                 │
    └────────┬────────┘
             門
```

### 4.4 工作室 (1.5坪) - 縮小版
- **位置**：中下區域
- **尺寸**：1.8m (寬) x 2.8m (深)
- **入口**：前側，統一風格推門
- **特點**：
  - 後牆窗戶（採光）
  - 書桌面向窗戶
  - 小型層架靠牆

**家具配置**：
```
      後牆 (窗戶)
    ┌───────────┐
    │   [窗]    │
    │  [書桌]   │
    │  [椅子]   │
    │           │
    │  [層架]   │
    └─────┬─────┘
          門
```

### 4.5 衛浴 (1.5坪)
- **位置**：右下區域
- **尺寸**：2.0m (寬) x 2.8m (深)
- **入口**：前側，統一風格推門
- **特點**：
  - 乾濕分離（玻璃隔間）
  - 洗手台靠近門口（方便使用）
  - 淋浴間在最裡面

**家具配置**：
```
         後牆
    ┌─────────────┐
    │   [淋浴]    │
    │  ─玻璃門─   │
    │             │
    │  [馬桶]     │
    │             │
    │[洗手台][鏡] │ ← 靠近門口
    └──────┬──────┘
           門
```

## 六、牆體配置

1. **外牆**：整個公寓外圍

3. **多功能室牆**：三面實牆 + 一面障子門
4. **衛浴牆**：四面實牆 + 一面有門
