## 三、空間連通關係

### 3.1 開放式區域 (公領域)
| 連通關係 | 說明 |
|---------|------|
| 客廳 ↔ 餐廚區 | **完全開放**，無隔間，視覺通透 |
| 客廳 ↔ 走道 | **開放**，無門，自然過渡 |

### 3.2 私密區域 (私領域)
| 空間 | 門的類型 | 位置 |
|------|---------|------|
| 主臥室 | Japandi 木框磨砂玻璃單扇門 | 走道上 X ≈ -0.75m |
| 工作室 / 多功能室 | 同上 | 走道上 X ≈ 2.2m |
| 衛浴 | 同上 | 走道上 X ≈ 4.3m |

### 3.3 門的統一規格 (Japandi 風格)
- **材質**：淺橡木框、單片磨砂玻璃、黑色簡約把手
- **寬度**：0.85m
- **高度**：2.1m
- **類型**：向內開啟的單扇門（統一門心落在 Z=2.0 的走道線上）

## 四、各空間詳細配置

### 4.1 客廳 (8坪) - 座標原點
- **位置 / 尺寸**：createLivingRoom(0, 0)，涵蓋 5.5m × 4.8m 公領域。
- **連通性**：右側完全向餐廚區打開，前方銜接 Z=2.0 走道。
- **採光**：左牆 X = -2.72m 安裝 2.4m × 2.2m 落地窗，並搭配紗簾與側簾。
- **家具配置**：
  - 沙發置於 (0, 0, 1.0)，旋轉 180° 面向北側電視牆（Z ≈ -2.15）。
  - 茶几與橢圓地毯置於公領域中心 (Z ≈ 0.0~0.5)，維持回字動線。
  - 電視櫃懸浮於後牆，附植栽、書籍、裝飾。
  - 角落配置落地植物 (-2.2, -1.9) 與邊桌 (1.8, 1.2)，不阻擋主要通道。

```
後牆 (Z = -2.4)
┌────────────────────┐
│   [電視牆+收納]      │
│                      │
│   ○ 茶几  (地毯)     │ → 右側開放到餐廚
│   ═══ 沙發 (朝北)    │
│                      │
│ 左牆整面落地窗       │
└────────────────────┘
        ↓ 走道 (Z = 2.0)
```

### 4.2 餐廚區 (4坪)
- **位置 / 尺寸**：createKitchenRoom(4.0, 0)，3.5m × 3.8m，與客廳共享地坪。
- **特點**：
  - L 型白色廚具沿後牆及右牆展開，長邊 2.4m 內含水槽、上櫃，短邊 0.6m 向室內延伸。
  - 檯面採淺木紋，搭配黑色把手與霧面水龍頭。
  - 中央配置 1.4m × 0.8m 木質餐桌，前後各兩張餐椅（總共四張），維持側向通道寬度。
  - 為保持 3D 視野，吊燈模組保留但不產生幾何（程式中返回空節點）。

```
後牆 (Z = -2.4)
┌──────────────┬────┐
│ 2.4m 長邊櫃體 │    │ ← 短邊轉向室內
│ [水槽][上櫃]  │ L  │
└──────────────┴────┘
      │ 1.4m 木質餐桌 │
      │ 前後各 2 張椅 │
```

### 4.3 主臥室 (5坪) - 擴大版
- **位置 / 尺寸**：createMasterBedroom(-0.75, 4.0)，約 4.0m × 4.0m。
- **配置**：
  - 床位於房間後半部 (局部座標 0.3, 0, 0.8)，床頭靠帶窗的後牆，整組旋轉 180° 讓床頭貼窗，床尾朝門。
  - 右側 (X ≈ 1.6) 只有一只床頭櫃與檯燈，維持左側通道寬度。
  - 2m 寬的隱藏把手衣櫃貼齊左牆後段 (X ≈ -1.3)，門板與牆面同色。
  - 化妝台 + 椅靠近門口的左牆前段 (X ≈ -1.3, Z ≈ -1.2)，前方配置化妝椅。
  - 後牆窗戶中心約在 (X ≈ -0.45, Z ≈ 5.9)，提供自然光。

```
後牆 (含窗)
┌────────────────────┐
│     [窗]             │
│      ═══ (床頭)      │
│                      │
│[衣櫃]      [化妝台+椅]│
└───────┬─────────────┘
        門 (X ≈ -0.75)
```

### 4.4 工作室 (1.5坪) - 縮小版
- **位置 / 尺寸**：createStudyRoom(2.2, 4.0)，1.8m × 2.8m。
- **配置**：
  - 書桌沿左牆擺放 (X ≈ -0.6) 並旋轉 90° 面向房間右側，方便左側窗戶採光灑下。
  - 後牆安裝磨砂玻璃窗 (Z ≈ 5.85) 提供柔和光線。
  - 顯示器位於桌面中央，辦公椅在室內中線靠門口側以維持轉身空間。
  - 層架靠近門口的左牆 (Z ≈ -1.0)；房間中央上方配置暖色點光源。

```
後牆 (窗)
┌────────────┐
│   [窗]        │
│[桌]→ (面向右) │
│[椅]           │
│[層架]         │
└──────┬────────┘
       門 (X ≈ 2.2)
```

### 4.5 衛浴 (1.5坪)
- **位置 / 尺寸**：createBathroom(4.3, 4.0)，2.0m × 2.8m。
- **配置**：
  - 由門口 (X ≈ 4.3) 進入後左側即為轉 90° 的懸浮洗手台與 LED 背光鏡，抽屜把手朝走道。
  - 馬桶置於房間中央靠左牆，與洗手台同向排列。
  - 最內側配置 0.9m × 0.9m 玻璃淋浴間，並用獨立玻璃滑門與頂軌打造乾濕分離。
  - 走道方向與淋浴之間另加一道磨砂玻璃隔間 (Z ≈ 0.6) 阻擋水氣。

```
後牆
┌──────────────┐
│    [淋浴間]     │
│ ──玻璃門 / 花灑 │
│                  │
│  [馬桶]          │
│[洗手台][鏡] ← 入門即到│
└──────────┬───┘
             門 (X ≈ 4.3)
```

## 六、牆體配置

### 需要的牆體：
1. **外殼**：X = -2.75 ~ 5.5、Z = -2.4 ~ 6.0 的外牆 + 地板 + 天花板。
2. **主臥前牆**：寬 4.0m，門洞置中 (X ≈ -0.75)；右側再立一面 4.0m 深的隔牆與工作室分界。
3. **工作室前牆 / 右牆**：前牆寬 1.8m（門置中），右側於 X ≈ 3.15m 立 4.0m 深實牆區隔衛浴。
4. **衛浴前牆**：寬 2.2m（門置中），室內另含玻璃隔間作乾濕分離。

### 不需要的牆體 (保持開放)：
1. 客廳與餐廚區之間（共用空間）
2. 客廳與走道之間（僅以高度 0 的地面線條界定）

### 不需要牆體 (開放)：
1. 客廳與餐廚區之間
2. 客廳與走道之間
